## Installation

1. Clone this repository.
```bash
git clone https://github.com/ATFinance/landing-page.git
cd landing-page/
```

2. Install package depedencies using npm.
```bash
npm install
```

## Available scripts

- `npm run dev`
- `npm run watch`
- `npm run hot`
- `npm run prod`
- `npm run serve`
